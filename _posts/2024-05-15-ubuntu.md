# Ununtu 相关

## 查看 ubuntu 硬件信息
1. 查看系统内核: uname -a
2. 查看 CPU: cat /proc/cpuinfo | grep model\ name
3. 查看内存: cat /proc/meminfo | grep MemTotal
4. 查看硬盘: df -lh
